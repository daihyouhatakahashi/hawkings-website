# Public Assets

このフォルダには以下のファイルを配置してください：

## 必須ファイル

- `favicon.png` - ブラウザタブ用アイコン (32x32px)
- `apple-touch-icon.png` - Appleデバイス用アイコン (180x180px)
- `og-image.jpg` - SNSシェア用画像 (1200x630px)
- `twitter-image.jpg` - Twitter用画像 (1200x675px)

## 推奨サイズ

### favicon.png
- サイズ: 32x32px または 16x16px
- 形式: PNG
- 背景: 透過または白

### apple-touch-icon.png
- サイズ: 180x180px
- 形式: PNG
- 角丸: 不要（自動適用される）

### og-image.jpg
- サイズ: 1200x630px
- 形式: JPG または PNG
- アスペクト比: 1.91:1

### twitter-image.jpg
- サイズ: 1200x675px
- 形式: JPG または PNG
- アスペクト比: 16:9

## 注意事項

これらの画像ファイルがない場合、SNSでシェアされた際にデフォルトの表示になります。
ブランドイメージのため、早急に作成・配置することを推奨します。
